## Deployment Options

| Option | Best For | Prerequisites |
|---|---|---|
| **Local (.NET SDK)** | Development, testing | .NET 10 SDK, Node.js 22+ |
| **Docker** | Production, CI/CD | Docker |
| **CI Pipeline** | Automated sync | GitHub Actions |
